Husk å flytte infobox under review.

Først snakke syntax

Regel, selector, attributt, verdi.
p {
    color: red;
}

Lage litt bs regler for å snakke om selectors

body {
    background-color: #c0fefe;
}

.infobox {
    border-style: solid;
}

Vise frem id tag (må lage id på posteren også)
#poster {
    height: 360px;
}

så float super quick:

float: right;

}
